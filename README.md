# AgroLens-AI
An end-to-end Computer Vision system utilizing Deep Learning to automate crop disease detection. This project implements custom CNN architectures and Transfer Learning to provide early-stage diagnostics, aimed at mitigating global yield loss and promoting sustainable agricultural practices.

### Quick Start

```bash
# Clone the repository
git clone https://github.com/code-with-bhumi/AgroLens-AI.git
cd AgroLens-AI
```
